# 项目技术栈

- 框架：Nuxt 4 + Nuxt UI 4
- 语言：TypeScript (严格模式)
- 样式：Tailwind CSS V4
- 状态管理：Pinia
- 数据库：Prisma + Sqlite3

# 编码规范

- 组件文件名使用PascalCase
- 工具函数使用camelCase
- 组件样式使用BEM规范
- TS不要使用any类型，而是使用unknown类型

# 我的编码偏好

- 缩进：2个空格
- 引号：双引号优于单引号
- 分号：always
- 对象和数组：不要尾随逗号
- 注释：使用JSDoc注释

# 我的习惯

- 优先使用const，其次let，避免var
- 函数优先使用箭头函数
- 解构赋值能用就用
- 条件渲染优先使用三元运算符

# 禁止事项

- 不需要启动开发服务器
- 不需要打开预览进行验证
